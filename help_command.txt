[Admin Mode]

exit             Exit Simuino

proj <name>      Read configuration file  		 
                 Enter "load" to load project

load <steps>     Load current proj and
                 create scenario of length <steps>
                 if no parameters, values are as configured

list             List available projects

run <steps>      Step until step=<steps>
                 If no steps given - enter Run Mode

reset            Reset simulation. Current step = 0

save  <name>     Save current configuration to <name>
                 If no <name>, saved to default configuration

info  <area>     Show info about <area>
    - scen       Show scenario data
    - conf       Display configuration
    - loop       Show loop information
    - error      Show Servuino errors
    - g++        Show compilation errors
    - help       Show command information

conf <par> <val> Edit configuration with commands below:
    - win        Set Window layout mode to <val> (0,1,2,3)
    - sketch     Set sketch source file
    - serv       Set Servuino data file
    - delay      Set delay to <val> (ms)
    - log        Set log level to <val> (0,1,2,3)
    - record     Record log to log.txt (1/0)
